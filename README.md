# Reinforcement Learning Project

Ce projet implémente et compare différentes méthodes d'apprentissage par renforcement pour résoudre les problèmes du projet Berkeley.

## Méthodes Implémentées

- **Value Iteration** : Méthode de programmation dynamique
- **Q-Learning Classique** : Apprentissage par différence temporelle
- **Approximate Q-Learning** : Avec features personnalisées

## Objectifs

- Répondre aux questions du projet Berkeley en ligne
- Comparer les performances des différentes méthodes
- Challenger les scores avec des features personnalisées

## Structure

Le projet contient l'implémentation des algorithmes et leur évaluation sur les environnements Berkeley, avec une analyse comparative des résultats obtenus.
